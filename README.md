### 🌲 Forest Survival
The flagship game in this repository. Survive in a forest, avoid the deer, build tools, and eventually hunt your way to victory.

#### Features:
- 3D Minecraft-style graphics (Three.js)
- Procedural sound engine (Web Audio API)
- Day/night cycle with enemy spawns (Aliens & Zombies)
- Crafting system and drivable vehicles

---

## 🛠 Development

This repository uses GitHub Actions to automatically build and deploy to GitHub Pages.

To run locally:
1. `cd forest_survival`
2. `npm install`
3. `npm run build`
4. `npm run serve`
5. Visit `http://localhost:3000`
