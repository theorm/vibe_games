# Forest survival

## Original game plot and description

Build a game. Here is the description:

### Summary

A "survival in a forest"  type of game. 
The player is the protagonist trying to survive in a forest inhabited by a vicious deer.
The deer's objective is to eat the player.
The player's objective is to  kill the deer. When the deer is killed, the game ends.

### Story

The game starts with the player on the edge of the forest with an axe. 
The forest is where the deer lives.  There is a circle in the middle of the forest the deer cannot enter. The deer cannot enter any deforested area.
The player has to chop trees to build tools for a house and eventually a house to survive while looking for the mines.
The mines is where the ore is. The player has to find the mines, mine ore and build a sword. The sword is the only weapon that can be used to chop the deer's head off. This is the only way to kill the deer.
There is a sequence of tools the player needs to build before he/she is able to make the sword.

### Style

Minecraft style. 3D game.
The player looks like a teenager with a floppy hat.
The deer staggers when he walks.

### Controls:

Keyboard. Arrows to move. Space to action.

### Sound

Deer makes scary yells.
Player burps, whistles and farts.
Background music: spooky with violin and disturbing ocarina
